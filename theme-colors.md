# Theme Colors - Mercedes Petronas

Personal color scheme inspired by Mercedes-AMG Petronas F1 team.

## Primary Palette

| Name | Hex | Usage |
|------|-----|-------|
| Petronas Teal | `#00D2BE` | Primary accent, directory, active elements |
| Dark Teal | `#00A896` | Secondary accent, git clean status |
| Silver | `#D8D8D8` | Tertiary, git modified/dirty status |

## Backgrounds (darkest to lightest)

| Name | Hex | Usage |
|------|-----|-------|
| Pure Black | `#0A0A0A` | Darkest backgrounds |
| Near Black | `#151515` | Status backgrounds |
| Dark Grey | `#1A1A1A` | Secondary panels, time |

## Text

| Name | Hex | Usage |
|------|-----|-------|
| Dark Text | `#161616` | Text on light/teal backgrounds |
| Light Text | `#f2f4f8` | Text on dark backgrounds |

## Accent Colors (unchanged from carbonfox)

| Name | Hex | Usage |
|------|-----|-------|
| Scarlet | `#CC2936` | Errors, conflicts |
| Green | `#25be6a` | Success indicators |
| Light Teal | `#7AEEE0` | Execution time |
